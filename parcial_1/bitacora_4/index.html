Notas de la clase

  • Cada elemento HTML tiene un valos de vizualizacion predeterminado segun el tipo de elemento que sea.
  • Un elemento con Display:block siempre comienza en una nueva linea y ocupa todo el ancho disponible, se exetiende hacia la izquierda y hacia la derecha tanto como sea posible.
  • Un elemento Inlineno comienza en una nueva linea y solo ocupa el ancho que sea necesario.
  • Display:inline-block permite establecer un acnho y una altura en un elemento, ademas se representan los margins y padding.

    • Resumen

    Display es de las mas complejas en el CSS 2.2. Display establece el tipo de la caja que genera cada elemento. La propiedad display nos permite editar de manera dinamica nuestro texto, posicionandolo en cajas, dandole personalizacion a los botones que queramos programar con javascript, entre otras posibilidades graficas.
