Notas de la clase

  • El header suele incluir elementos de titulo de la pagina, logos, etc
  • Un inicio puede incluir el logotipo e informacion relevante
  • El header tambien puede incluir un acerca de para referenciar la informacion del footer
  • Contacto puede tambien referenciar los contactos dentro del footer, o incluirlos en el header

    • Resumen

    La etiqueta <'header> en HTML se utiliza para contener información de encabezado de un documento o sección. El contenido dentro de la etiqueta <'header> suele incluir elementos como el título de la página, el logotipo, la navegación y otros elementos de encabezado relacionados.
