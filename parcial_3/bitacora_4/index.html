Ideas Clave

    ::after
    ::before
    content

Notas de la clase

  • El atributo content se utiliza para añadir contenido antes o despues del elemento
  • ::after se utiliza para cambiar afectar los elementos posteriores
  • ::before se utiliza para cambiar afectar los elementos anteriores

    • Resumen

    En CSS, ::before crea un pseudoelemento que es el primer hijo del elemento seleccionado. Es usado normalmente para añadir contenido estético a un elemento, usando la propiedad content. Este elemento se muestra en línea con el texto de forma predeterminada.
